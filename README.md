📌 About
A fully functional online voting portal for Indian elections — connected to Aadhaar (UIDAI), Election Commission of India database, and DigiLocker. Every vote is AES-256 encrypted and stored immutably on blockchain. Fully responsive on desktop and mobile.
✨ Features
🪪 Aadhaar OTP + Biometric verification via UIDAI
🏛️ Election Commission database — voter eligibility & constituency lookup
🔐 AES-256 encryption + Zero Knowledge Proof (ZKP) anonymity
⛓️ Blockchain storage — immutable, tamper-proof vote ledger
📜 Smart Contract — auto vote counting, one-vote enforcement
🗳️ 7-step voting flow — Login → OTP → Biometric → Ballot → Confirm → Blockchain → Receipt
🚫 NOTA support (None of the Above)
🤖 AI fraud detection simulation
📱 Fully responsive — works on desktop and mobile
